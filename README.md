# 비움 (VIUM)

## 프로젝트 개요

비움(VIUM)은 전기차 충전소 이용 과정에서 발생하는 장기 점유, 충전 대기, 충전기 고장 정보 부족 등의 문제를 해결하기 위한 스마트 충전 관리 시스템입니다.

본 프로젝트는 지도 기반 충전소 탐색, 사용자 참여형 충전기 상태 공유, 마일리지 보상 시스템, 하드웨어 기반 출차 검증 기능을 통합하여 전기차 충전소의 이용 효율을 높이는 것을 목표로 합니다.

또한 Arduino와 카메라 비전 기술을 활용하여 충전 커넥터 연결 상태와 차량의 실제 출차 여부를 교차 검증함으로써, 단순한 정보 제공을 넘어 실제 현장 상황과 연동되는 O2O 기반 충전 관리 플랫폼을 구현합니다.

## 개발 배경

전기차 보급이 확대되면서 충전 인프라 역시 증가하고 있지만, 여전히 공용 충전소 부족, 충전 완료 후 장기 점유, 충전기 고장으로 인한 헛걸음 문제가 발생하고 있습니다.

기존 전기차 충전 서비스는 주로 충전소 위치 안내와 결제 기능에 집중되어 있어, 실제 현장에서 발생하는 장기 점유나 충전기 고장 상태를 실시간으로 반영하기 어렵다는 한계가 있습니다.

비움은 이러한 문제를 해결하기 위해 사용자 행동을 유도하는 보상 시스템과 하드웨어 기반 검증 시스템을 결합하여, 충전소 회전율 향상과 데이터 신뢰성 확보를 동시에 달성하고자 합니다.

## 주요 기능

### 1. 지도 기반 충전소 탐색 및 요금 비교

사용자는 지도 화면을 통해 주변 충전소의 위치, 충전기 규격, 요금 정보를 확인할 수 있습니다.

충전기 유형별 요금을 직관적으로 제공하여 사용자가 합리적으로 충전소를 선택할 수 있도록 지원합니다.

### 2. 사용자 참여형 충전기 상태 제보

사용자는 충전기 고장, 충전 실패, 기기 파손 등의 상태를 키워드와 사진으로 제보할 수 있습니다.

이를 통해 공공데이터 API만으로 파악하기 어려운 현장 정보를 보완하고, 충전기 상태 정보의 신뢰성을 높입니다.

### 3. 마일리지 보상 시스템

충전 완료 후 일정 시간 내 차량을 이동하거나, 유효한 충전기 고장 제보를 남긴 사용자에게 마일리지를 지급합니다.

적립된 마일리지는 추후 충전 요금 결제 시 사용할 수 있도록 설계합니다.

### 4. 80% 충전 제한 유도

배터리 안전을 위해 80% 충전 제한 기능을 제공하고, 이를 선택한 사용자에게 추가 마일리지를 지급합니다.

이를 통해 배터리 과충전 위험을 줄이고 안전한 전기차 이용 문화를 유도합니다.

### 5. 하드웨어 기반 출차 검증

Arduino 기반 센서를 통해 충전 커넥터의 연결 상태를 감지하고, Raspberry Pi와 OpenCV 기반 카메라 시스템을 통해 차량의 실제 출차 여부를 확인합니다.

커넥터 분리 신호와 차량 이탈 신호가 모두 확인되었을 때 정상 출차로 판단하여 부정 마일리지 획득을 방지합니다.

### 6. V2G 확장 구조

전기차를 단순한 이동 수단이 아닌 에너지 자원으로 활용할 수 있도록 V2G(Vehicle to Grid) 개념을 반영합니다.

향후 전력 판매 및 스마트 에너지 플랫폼으로 확장 가능한 구조를 목표로 합니다.

## 시스템 구성

```text
VIUM
├── vium-web
│   └── 프론트엔드 화면 및 사용자 인터페이스
├── vium-backend
│   └── 백엔드 API, 데이터 처리, 사용자 및 충전소 관리
└── hardware
    ├── arduino
    │   └── 배터리 및 충전 커넥터 상태 감지
    └── raspberrypi
        └── OpenCV 기반 차량 감지 및 실시간 스트리밍
```

## 차별성

기존 전기차 충전 서비스가 충전소 위치 안내와 결제 중심의 단방향 서비스에 머물렀다면, 비움은 사용자 행동을 유도하는 마일리지 보상 시스템과 하드웨어 기반 출차 검증 기능을 결합한 통합형 서비스입니다.

특히 커넥터 연결 상태와 차량의 실제 출차 여부를 함께 확인하는 이중 검증 구조를 통해, 단순히 충전이 종료되었는지뿐만 아니라 실제로 차량이 이동했는지까지 판단할 수 있습니다.

또한 사용자 참여형 고장 제보 기능을 통해 공공데이터의 한계를 보완하고, 실시간 현장 데이터를 기반으로 보다 신뢰도 높은 충전소 정보를 제공합니다.
